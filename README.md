# Ola-Data-Analyst

# 🚖 OLA Ride Booking Analysis

## 📌 Overview

The **OLA Ride Booking Analysis** project is an end-to-end Data Analytics project focused on analyzing ride booking data to uncover business insights and improve operational performance. The project covers the complete analytics workflow—from data loading and cleaning in Python to SQL analysis, interactive Power BI dashboards, and business reporting.

---

## 📊 Dataset

- **Dataset:** OLA Ride Booking Dataset
- **Rows:** 103,024
- **Columns:** 20
- **Domain:** Ride Booking & Transportation
- **Database:** MySQL Workbench

---

## 🛠️ Tools & Technologies

- **Programming:** Python
- **Libraries:** Pandas, NumPy, Matplotlib, Seaborn
- **Database:** MySQL Workbench, SQL
- **Visualization:** Power BI
- **Reporting:** Gamma
- **Version Control:** Git, GitHub

---

## 🔄 Project Workflow

1. Loaded the dataset using Python.
2. Performed Exploratory Data Analysis (EDA).
3. Cleaned and validated the dataset.
4. Imported data into MySQL Workbench.
5. Wrote SQL queries to extract business insights.
6. Built interactive Power BI dashboards.
7. Prepared a business report and presentation using Gamma.

---

## 📈 Dashboard Highlights

- Booking Overview
- Revenue Analysis
- Booking Status & Cancellation Analysis
- Vehicle Performance
- Customer & Driver Ratings
- Payment Method Analysis
- Key Business KPIs

---

## 📌 Key Results

- Identified booking trends and cancellation patterns.
- Analyzed revenue across vehicle types and payment methods.
- Evaluated customer and driver ratings.
- Generated actionable business recommendations through interactive dashboards.

---

## ▶️ How to Run

1. Clone this repository.
2. Install the required Python libraries.
3. Load the dataset into Python.
4. Perform data cleaning and EDA.
5. Import the cleaned data into MySQL Workbench.
6. Execute the SQL queries.
7. Open the Power BI dashboard (`.pbix`) to explore insights.
